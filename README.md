# 固件简要说明：

固件每周一早上4点自动编译。

固件信息里的时间为编译开始的时间，方便核对上游源码提交时间。

MEDIATEK系列。

# 目录简要说明：

Depends.txt——环境依赖列表

Uboots.sh——Uboot型号脚本

workflows——自定义CI配置
